# Astrocric Admin Panel

React-based admin panel for managing Astrocric platform.

## Quick Start

1. **Install dependencies**
   ```bash
   npm install
   ```

2. **Start development server**
   ```bash
   npm run dev
   ```

3. **Build for production**
   ```bash
   npm run build
   ```

Admin panel runs on http://localhost:5173
