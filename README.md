# Astrocric

Cricket prediction platform with Flutter mobile app, Node.js backend, and React admin panel.

## Project Structure

```
astrocric/
├── backend/          # Node.js + Express backend
├── mobile/           # Flutter mobile app
├── admin-panel/      # React admin panel
└── README.md
```

## Tech Stack

- **Mobile**: Flutter (Android)
- **Backend**: Node.js + Express + PostgreSQL
- **Admin**: React.js
- **Payment**: PhonePe Business (Zero commission)
- **Hosting**: Railway VPS
- **SMS**: MSG91
